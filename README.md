## Description:

This app is a forum specific for Mouse Quest that will allow players to create and see builds that more experienced players have made. 

## User Story

As A user of the Mouse Quest game
<br>
I WANT to create and see builds
<br>
SO THAT I can become better at the game.

<p align="right">(<a href="#top">back to top</a>)</p>

## Application Features:

- Variety of build options
- View posts by username and view all posts
